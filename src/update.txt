Escape Distance:
1. File -> Escape Distance...
2. The user will be prompted to enter in a number greater than 0.
		Rejected inputs:
		- Numbers less than 0
		- Empty input
		- Anything that is not a number (e.g characters, symbols, ect.)

Escape Time:
1. File -> Escape Time...
2. The user will be prompted to enter in an integer greater than 0.
		Rejected Inputs:
		- Numbers less than 0
		- Numbers with decimals
		- Empty Input
		- Anything that is not a number (e.g characters, symbols, ect.)
